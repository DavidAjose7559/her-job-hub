# ✨ Her Job Hub

A full AI-powered job application assistant. Generate tailored resumes and cover letters, track every application, and auto-fill job forms with a Chrome extension.

## Project Structure

```
her-job-hub/
├── app/              # React + Vite web app
├── extension/        # Chrome extension (Manifest V3)
├── .env.example      # Copy to .env and add your API key
└── README.md
```

## Web App Features

- **Profile** — Save your resume, skills with expertise levels, job preferences, and a writing sample the AI uses to match your tone
- **Apply** — Paste any job description and get a tailored resume + cover letter generated by Claude (Sonnet 4.6) in real-time
- **Tracker** — Add, update, and delete job applications with statuses (Saved → Applied → Interview → Offer / Rejected) and a stats dashboard
- All data persisted in `localStorage` — no account needed

## Chrome Extension Features

- Syncs your profile from the web app automatically
- Detects form fields on LinkedIn, Indeed, Greenhouse, Lever, and Workday
- One-click auto-fill for name, email, phone, LinkedIn, cover letter, and more

## Setup

### 1. Install dependencies

```bash
cd app
npm install
```

### 2. Set your Anthropic API key

```bash
cp .env.example .env
# Edit .env and set your key:
# VITE_ANTHROPIC_API_KEY=sk-ant-...
```

Get your key at [console.anthropic.com](https://console.anthropic.com).

> **Note:** The API key is loaded client-side via `VITE_` prefix and will be visible in the browser. This is fine for personal use. For a production app, proxy the API call through a backend.

### 3. Run the dev server

```bash
cd app
npm run dev
# → http://localhost:5173
```

### 4. Load the Chrome extension

1. Open `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked** → select the `/extension` folder
4. After saving your profile in the web app, the extension will sync it automatically

## Deploying to Vercel

### Option A — Vercel CLI

```bash
npm install -g vercel
cd app
vercel
# Follow the prompts. Set VITE_ANTHROPIC_API_KEY in the Vercel dashboard under Settings → Environment Variables.
```

### Option B — GitHub + Vercel Dashboard

1. Push to GitHub (see below)
2. Go to [vercel.com](https://vercel.com) → **Add New Project**
3. Import your GitHub repo
4. Set **Root Directory** to `app`
5. Add environment variable `VITE_ANTHROPIC_API_KEY`
6. Click **Deploy**

After deploying, update the extension:
- In `extension/popup.js` → change `APP_URL` to your Vercel URL
- In `extension/manifest.json` → add your Vercel domain to `host_permissions`
- Reload the extension in `chrome://extensions`
